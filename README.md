# Build

## Windows

### Requirements

- MSVC 2017 x64
- Qt > 5.12.1 (for MSVC 2017 x64)
- AutoHotkey (for tests)

### Process

- Set QTDIR environment variable
- Run MCVS console
- Call silverkey/deploy/build.bat
- Wait
- Silverkey will deployed to silverkey/bin directory
